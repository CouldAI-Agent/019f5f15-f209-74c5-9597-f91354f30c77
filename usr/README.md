# BGMI Pro Settings: Samsung A70s

A comprehensive Flutter application providing the best Battlegrounds Mobile India (BGMI) settings, control layouts, and display optimizations tailored specifically for the Samsung Galaxy A70s smartphone.

## Features
- **Optimal Sensitivity Settings**: Recommended values for Camera, ADS, and Gyroscope settings for accurate aiming and recoil control.
- **Pro Control Layouts**: Guides for 3-finger and 4-finger claw setups optimized for the A70s's large 6.7-inch display.
- **Display Optimization**: Specific phone and game graphics settings to maximize the Super AMOLED panel's potential while maintaining smooth 60fps gameplay.

## Tech Stack
- Flutter
- Dart
- Material Design 3

## Setup Instructions
1. Ensure you have the Flutter SDK installed.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application.
